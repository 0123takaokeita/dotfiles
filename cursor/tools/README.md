# 汎用ツール（tools）

メインエージェントや agents が共通で参照する「ツールの使い方」や「手順」を定義する場所です。言語・フレームワークに依存しない汎用的な内容を置きます。

## 役割

- コードベース検索・grep の使い方やクエリの立て方
- ターミナル実行の前提（例: テスト・Lint の実行手順）
- プロジェクト種別（Laravel / フロントなど）に依存しない汎用手順

## 形式

- Markdown（`.md`）が中心
- 必要に応じて `tools/` 配下に小さいスキルを置くことも可

## 参照のしかた

- 各 agents の .md の **references** で、参照すべきツールのパスを指定する
- 既存の `commands/` や `skills/` から「○○するときは `cursor/tools/xxx.md` を参照せよ」と書く
